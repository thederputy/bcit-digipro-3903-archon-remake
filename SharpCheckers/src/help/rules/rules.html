Official Game Rules

I thought that it would be a good idea to provide the official game rules for checkers (draughts).

I have taken the liberty to use the rules that were described in the web site Game Cabinet.

As stated on that web site, these rules were found on the help system of an old MS-DOS checkers game.

Because of that, no page on this section has a copyright statement. The copyright belongs to the original author that I failed to find.
